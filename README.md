# Provenance Key

Online gaming taught us a lesson: when bots become indistinguishable from players, trust must be earned through evidence.

AI is bringing that lesson to reality.

## Anti-Cheat for Reality

Provenance Key is an open-source protocol for:

- Scene Inspection Hashes
- Digital Provenance
- Creator Attribution
- Derivative Tracking
- LLM Attribution
- Creator Compensation
- Evidence-Based Authenticity

## Core Principles

- Open Source
- Runs on Existing Hardware
- No Central Authority
- Trust Through Evidence
- Attribution Preserved Forever
- Derivatives Tracked Forever
- Creators Get Paid

## Key Ideas

### The Hash Starts Before the Shot

### Git for Media
